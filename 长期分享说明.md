# 长期分享说明

## 现在这套怎么用

你现在可以用这个固定外链子域名来分享主页：

- `https://song-smp-homepage.loca.lt/generated-views/visitor-homepage.html`

这套方式的特点是：

- 地址固定，发给别人不会变
- 你本地改主页、改照片墙、改案例库后，这个地址里看到的内容也会一起更新
- 但前提是你的电脑要开着，而且公开分享服务要启动着

## 启动分享

在项目目录里运行：

```bash
./scripts/start-public-share.sh
```

启动后会输出：

- 游客主页地址
- 访客名单后台地址

## 查看当前分享地址

```bash
./scripts/show-public-share.sh
```

## 关闭分享

```bash
./scripts/stop-public-share.sh
```

## 适合你的使用方式

如果你只是平时偶尔更新内容、偶尔发给顾客看，这套已经够用：

1. 打开电脑
2. 运行 `./scripts/start-public-share.sh`
3. 把游客主页链接发给别人
4. 你更新案例图、工作照、文案后，刷新外链页面就能看到最新内容

## 真正永久稳定的版本

如果你想做到：

- 电脑关机后别人也能访问
- 不需要每次手动启动
- 真正长期稳定给客户发

那下一步需要把站点正式部署到云端。

你这套目前带有：

- 本地上传案例图
- 本地登记联系方式
- 本地名单后台

所以要想“完全永久”，我下一步建议给你做：

1. 一个云端静态站点
2. 一个云端数据库/表单接口
3. 一个固定域名

这样才是彻底长期可访问。
